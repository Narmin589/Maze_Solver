# Maze_Solver 

A Python program that finds a path through a maze using the recursive algorithm.

## How it works
- Reads maze layout and configuration from a text file
- Accepts user input for the starting position
- Navigates right, down, left, or up — avoiding walls
- Prints the solution path when found

## How to run
1. Clone the repository
2. Run: `python maze_solver.py`
3. Enter your starting position when prompted
4. Maze configurations are provided in `maze1.txt` and `maze2.txt`

## Technologies used
- Python
- Recursive algorithm
- File handling
